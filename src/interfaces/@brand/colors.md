The MRA palette is built for a dark, editorial, confidently utilitarian product. Asphalt is the dominant mood — a near-black with warmth, not a flat OLED void. Mojo Red is the single spark of brand, used with discipline. Paper is a context-specific warm cream surface that appears only when something represents a student's own written work (artifact panels, printed deliverables) — it is not a light-mode theme.

```colors
Asphalt:
  value: "#0B0B0C"
  description: Base canvas. App background. Near-black with barely-warm undertone.
Ironwood:
  value: "#141416"
  description: First elevation. Panels, side rails, nav surfaces.
Gunmetal:
  value: "#1C1C1F"
  description: Second elevation. Inputs, chat composer, user message bubble, cards inside panels.
Graphite:
  value: "#27272B"
  description: Hairline borders and dividers on dark surfaces.
Paper:
  value: "#F2EDE3"
  description: Warm cream surface. Artifact panels, printed deliverables, long-form editable output.
Newsprint:
  value: "#E6DFD0"
  description: Paper's divider sibling. Rules and subtle blocks on Paper surfaces.
Mojo Red:
  value: "#D4262C"
  description: The single brand accent. Primary CTA, active state, Mr. A monogram, key selection.
Ember:
  value: "#A81C22"
  description: Pressed/hover state of Mojo Red. Also used for destructive confirmation.
Clay:
  value: "#F26F68"
  description: Lightened red tint for small moments where full Mojo Red would overpower — tags, inline highlights, active dots.
Bone White:
  value: "#F4EFE6"
  description: Primary text on dark surfaces. Warm off-white, never pure white.
Dust:
  value: "#A29C90"
  description: Secondary text, metadata, captions on dark surfaces.
Smoke:
  value: "#6C675E"
  description: Tertiary text, placeholders, timestamp marginalia.
Coal:
  value: "#1A1815"
  description: Primary text on Paper surfaces. Warm near-black.
Brass:
  value: "#B88B3E"
  description: Success state. Completed-step markers. Muted gold, not green.
Rust:
  value: "#C2582A"
  description: Warning state. Burnt orange, earthy.
```
